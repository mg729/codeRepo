## How to Code
1. 문제분석  
    - 수치 및 조건 정리하기(예외처리)   
    - 전체적인 흐름 그리기  
    - 입출력 예제 이해하기  
2. 모델링  
    - 필수 알고리즘은 암기  
3. 절차적 사고  
4. 구현  

## Error analyzing
- 제한 및 대소관계(이상, 이하, 초과, 미만, min, max)  
- 예외처리 (단, 없는 경우는 ~~)  
- 입력과 출력 (공백, 양식, 순서, 정렬 유무)  
- 시간 제한과 메모리 제한  
- 알고리즘이 맞는가?  
- 내가 생각한 로직대로 구현했나?  
- 불필요한 반복문이 있는가?  
- 중복은 처리했는가?
